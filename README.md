## Description

情報処理部、通称『JS部』の部室では、瀬尾絵子と篠宮樹里の二人の部員が今日もJavaScriptについてゆるい会話を繰り広げます。恋に落ちそうなコードを求めて。

登場人物

瀬尾絵子（せのおえこ）　JS部部長（押し付けられた）。主にツッコミ担当。  
篠宮樹里（しのみやじゅり）　JS部副部長。JavaScriptその他諸々について絵子の良き先生。
